MechSolutions Landing Page

MechSolutions is a responsive, modern landing page for a mechanical engineering and automation solutions company. It showcases services, products, automation solutions, testimonials, and provides a call-to-action for potential clients. Built with React, Vite, and TailwindCSS.

Features

Fully responsive design

Hero section

Stats section highlighting projects, experience, and industries served

About section with text and image

Services section with sliding cards

Automation solutions section with image and description

Products gallery with images and names

Testimonials carousel (3 visible at a time)

Call-to-action section

Functional navbar linking to all sections

Footer with contact info and quick links

Technologies Used

React 19 – Frontend framework

Vite 7 – Build tool

TailwindCSS 4 – Styling framework

JavaScript ES6+ – Logic and interactivity

Installation

Clone the repository:

git clone https://github.com/paulthuku8419-eng/PAUL-THUKU.git
cd PAUL-THUKU


Install dependencies:

npm install


Start the development server:

npm run dev


Build for production:

npm run build

Deployment

The site can be deployed on Vercel, Netlify, or GitHub Pages.

For Vercel: Set Build command to npm run build and Output directory to dist.

Usage

Navigate using the navbar links to different sections.

Click the CTA buttons to contact the company.

Services and testimonials can be navigated using the Next/Previous buttons.
